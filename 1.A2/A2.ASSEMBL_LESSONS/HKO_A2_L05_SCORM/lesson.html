📖 Useful Language · Stage 2The Language Box

Future with Going To

We use this to describe finished HR actions
FormStructureHR Example
✅ +subject + verb+edI started in HR five years ago.
❌ −subject + didn't + base verbShe didn't work in payroll.
❓ ?Did + subject + base verb?Did you work there long?
⚠ Common Error
didn't worked → didn't work ✓
