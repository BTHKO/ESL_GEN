## LESSON STRUCTURE (12 Lessons Total)

Each lesson follows **7-Stage Scaffolding Model**:

1. **Visual Kickoff** (Exposure) - 12-18 min
2. **Prepping for Success** (Introduction) - 15-20 min
3. **Strategic Activation** (Exploration) - 12-15 min
4. **Main Input & Comprehension** (Understanding Activity 1) - 25-35 min
5. **Language Discovery & Practice** (Understanding Activities 2-3) - 30-40 min
6. **Professional Application** (Free Practice) - 20-30 min
7. **Lesson Wrap** (Reflection) - 10-15 min

**Each lesson includes:**
- Vocabulary lesson (14 stages total: 7 vocab + 7 grammar)
- Minimum 10 interactive games (5 vocab, 5 grammar)
- Downloadable materials
- Self-assessment tools

## METHODOLOGY

**E-Learning Asincrónico Nivel 2 Characteristics:**

1. **Self-Paced Learning**
   - Participants advance at own pace within deadlines
   - Flexible scheduling (24/7 platform access)
   - Recommended 10 hours/week, adjustable

2. **Interactive Multimedia**
   - Videos, audio recordings, images
   - Gamified activities (matching, drag-drop, quizzes)
   - Web Speech API for pronunciation practice
   - Simulations and role-plays

3. **Asynchronous Tutor Support**
   - Responds within 48 hours maximum
   - Reviews written/spoken assignments
   - Provides personalized feedback
   - Accessible via platform messaging/forums

4. **Task-Based Learning (TBLT)**
   - Every lesson culminates in authentic professional task
   - Language taught to accomplish real-world goals
   - Focus on meaning first, form second

5. **Discovery Learning**
   - Students notice patterns before explicit instruction
   - Guided discovery with scaffolding
   - Grammar emerges from authentic examples

6. **Skills Integration**
   - All 4 skills practiced in every lesson
   - Skills rotate and build on each other
   - Resources reused for vocabulary and grammar instruction

## PART A: VOCABULARY LESSON (7 Stages)

### ══════════════════════════════════════════════════════════════
### **STAGE 1: VISUAL KICKOFF** (Exposure)
### ══════════════════════════════════════════════════════════════

**Duration:** 12-18 minutes  
**Purpose:** Encounter words naturally within authentic context  
**Cognitive Load:** LOW (Recognition only)

**Materials:**
- 1-2 contextual images (IMG-L##-S01-VOC)
- 2-3 observation questions

**Activity:**
```
┌─────────────────────────────────────────────────────────────────┐
│  [CONTEXTUAL IMAGE showing HR scenario with target vocab]        │
│                                                                   │
│  🔍 OBSERVATION QUESTIONS:                                       │
│  1. What's happening in this scene?                              │
│  2. What do you notice about [specific element]?                 │
│  3. Have you experienced something similar in your work?         │
│                                                                   │
│  [A1-A2 Spanish translations provided]                           │
└─────────────────────────────────────────────────────────────────┘
```

**Learner Task:** Observe, guess meanings from visual context  
**Outcome:** Curiosity activated, prior knowledge tapped  
**No Assessment:** Purely exploratory

---

### ══════════════════════════════════════════════════════════════
### **STAGE 2: PREPPING FOR SUCCESS** (Introduction via Games)
### ══════════════════════════════════════════════════════════════

**Duration:** 15-20 minutes  
**Purpose:** Playful interaction with words; build confidence  
**Cognitive Load:** LOW-MEDIUM (Guided practice)

**GAME 1: VOCABULARY MATCHING** (8-10 pairs)
- Click-to-match word-meaning or word-image pairs
- All HR context
- Feedback: Green/red + sounds

**GAME 2: WORD ORDER CHALLENGE** (6-8 sentences)
- Drag words to build sentences using target vocabulary
- Snap-in effects
- Feedback: Shake/success animations

**Optional:** Pronunciation practice (Web Speech API)

**Outcome:** Confidence with word forms and meanings  
**Assessment:** Game scores tracked (formative, not graded)

---

### ══════════════════════════════════════════════════════════════
### **STAGE 3: STRATEGIC ACTIVATION** (Exploration)
### ══════════════════════════════════════════════════════════════

**Duration:** 12-15 minutes  
**Purpose:** See words in varied authentic contexts  
**Cognitive Load:** MEDIUM (Meaning-making)

**Materials:**
- 1 new image (different HR scenario, IMG-L##-S03-VOC)
- "Word Window" reference box (definitions + examples)
- 3-4 exploration questions

**Activity:**
```
┌─────────────────────────────────────────────────────────────────┐
│  📖 WORD WINDOW                                                  │
│  ────────────────────────────────────────────────────────────   │
│  • [Word 1]: [Definition in context]                            │
│    Example: "[HR sentence]"                                     │
│  • [Word 2]: [Definition in context]                            │
│    Example: "[HR sentence]"                                     │
│  ...                                                             │
│                                                                   │
│  🇪🇸 [Spanish translations for A1-A2]                           │
└─────────────────────────────────────────────────────────────────┘

EXPLORATION QUESTIONS:
1. How would you use [word] in your work?
2. Which words are related? Why?
3. Can you think of a situation where you'd say [word]?
```

**Outcome:** Meaning clarity, contextual understanding  
**Assessment:** Question responses (formative)

---

### ══════════════════════════════════════════════════════════════
### **STAGE 4: MAIN INPUT & COMPREHENSION** (Understanding Activity 1)
### ══════════════════════════════════════════════════════════════

**Duration:** 25-35 minutes  
**Purpose:** Comprehend words in extended authentic discourse  
**Cognitive Load:** MEDIUM-HIGH (Comprehension + analysis)

**Input Type:** [Odd lessons = LISTENING | Even lessons = READING]

**Materials:**
- Audio (1.5-3 min) OR Reading text (150-500 words)
- Target vocabulary HIGHLIGHTED in text/script
- 1 context image (IMG-L##-S04-VOC)
- Comprehension activities (3 types)

**Comprehension Activities:**

1. **Word Identification** (5 questions)
   - "Which word in the text means...?"
   - Multiple choice or matching

2. **Word Usage Matching** (5 items)
   - Match sentences from text to definitions

3. **Contextual Understanding** (5 questions)
   - "What did they decide to do?"
   - "Why did she use the word [X]?"

**Downloadable:** Audio script + Spanish translation (A1-A2) + pronunciation guide

**Outcome:** Words understood in extended professional context  
**Assessment:** Comprehension scores (20% of formative grade)

---

### ══════════════════════════════════════════════════════════════
### **STAGE 5: ACTIVE PRACTICE & APPLICATION** (Understanding Activities 2-3)
### ══════════════════════════════════════════════════════════════

**Duration:** 30-40 minutes  
**Purpose:** Produce language using target vocabulary  
**Cognitive Load:** HIGH (Controlled production)

**Discovery Activity:**
```
❓ VOCABULARY PATTERNS
Look at these groups of words. What do they have in common?

Group 1: recruit → recruitment → recruiter
Group 2: apply → application → applicant
Group 3: interview → interviewee → interviewer

What pattern do you notice?
```

**Pattern Box:** Vocabulary relationships (word families, collocations)

**GAME 1: VOCABULARY IN SENTENCES** (5-6 gaps)
- Drag word into sentence gap
- HR contexts

**GAME 2: WORD APPLICATION QUIZ** (6-8 questions)
- Select correct sentence using target word
- Multiple choice with feedback

**GAME 3: FILL-IN-THE-BLANK** (5-6 items)
- Type or select word
- Contextual clues provided

**Optional:** Record yourself using words in sentences

**Downloadable:** Vocabulary reference sheet + word forms table

**Outcome:** Produce words accurately in varied contexts  
**Assessment:** Game accuracy (25% of formative grade)
