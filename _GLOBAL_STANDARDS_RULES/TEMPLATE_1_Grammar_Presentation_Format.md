**Purpose:** Standardized format for presenting grammar concepts across all CEFR levels

## 📋 USAGE INSTRUCTIONS

**When to Use This Template:**
- Every grammar lesson (Stage 5: Language Discovery & Practice)
- Grammar reference downloadables
- Grammar summary boxes throughout lessons

**How to Customize:**
1. Replace [GRAMMAR CONCEPT] with specific structure (e.g., "Present Perfect")
2. Fill in all bracketed sections with level-appropriate content
3. Ensure Spanish translations for A1-A2 levels

```
┌─────────────────────────────────────────────────────────────────┐

│                                                                  │
│  1. "[Example sentence 1 with target grammar BOLDED]"          │
│  2. "[Example sentence 2 with target grammar BOLDED]"          │
│  3. "[Example sentence 3 with target grammar BOLDED]"          │
│                                                                  │
│  ❓ DISCOVERY QUESTIONS:                                        │
│  • What do you notice about the [BOLDED/highlighted] parts?    │
│  • What stays the same? What changes?                           │
│  • When do you think we use this form?                          │
│                                                                  │
│  [A1-A2 Spanish translation]:                                   │
│  • ¿Qué notas sobre las partes [en negrita/resaltadas]?        │
│  • ¿Qué se mantiene igual? ¿Qué cambia?                        │
│  • ¿Cuándo crees que usamos esta forma?                         │
└─────────────────────────────────────────────────────────────────┘
```

**Design Notes:**
- 1 image that shows grammar in authentic HR context
- 3 example sentences minimum (all from HR scenarios)
- Discovery questions guide pattern recognition
- NO explicit rule yet (learners discover first)

## 🎨 DESIGN SPECIFICATIONS

### **Colors & Branding:**
- Primary: `#d2691e` (burnt orange) - Headers, buttons, accents
- Secondary: `#4a4a4a` (gunmetal gray) - Text, borders
- Accent: `#8b4513` (saddle brown) - Highlights, callouts
- Success: `#4caf50` (green) - Correct answers, checkmarks
- Error: `#f44336` (red) - Incorrect answers, warnings
- Background: `#ffffff` (white) / `#f9f9f9` (light gray)

### **Typography:**
- Primary Font: 'Aptos', 'Calibri', 'Segoe UI', sans-serif
- Headings: Bold, 18-24pt
- Body Text: Regular, 14-16pt
- Instructions: Italic, 12-14pt
- Spanish Translations: Italic, gray color, 12pt

### **Icons & Emojis:**
- 🔍 Discovery/Exploration
- 📘 Reference/Information
- ⚙️ Structure/Form
- 💡 Meaning/Understanding
- ⏰ Time/Duration
- ⚠️ Warning/Common Mistake
- ✅ Correct/Affirmative
- ❌ Incorrect/Negative
- ❓ Question
- 🔊 Audio/Sound
- 🎯 Game/Activity
- 📥 Download
- 🎉 Success/Completion

### **Layout Principles:**
- Clean, uncluttered design
- White space for readability
- Consistent spacing (margins, padding)
- Mobile-responsive (works on phones, tablets, desktop)
- Print-friendly (clear page breaks, readable fonts)

## 🔄 REUSABILITY NOTES

**Create Once, Use Many Times:**
- Grammar reference sheets are reusable across multiple lessons teaching same concept
- Practice exercise banks can be adapted with new contexts
- Audio script format is template (content changes, structure stays same)
- Games can be repopulated with new sentences/questions

**Modular Design:**
- Each section can stand alone
- Mix and match sections based on lesson needs
- Sections can be reordered if pedagogically justified
- Templates can scale up/down for different CEFR levels

- Game completion rates (target: 90%+)
