# TEMPLATE 2: VOCABULARY GLOSSARY FORMAT
## Master Template for Vocabulary Presentation

**Version:** 1.0 | **Date:** February 2026  
**Purpose:** Standardized format for presenting vocabulary across all CEFR levels  
**Context:** HR Professional ESL - SENCE Asynchronous E-Learning

---

## 📋 USAGE INSTRUCTIONS

**When to Use:**
- Stage 2: Prepping for Success (vocabulary warm-up)
- Stage 3: Strategic Activation (vocabulary reference box)
- Stage 5: Active Practice (vocabulary pattern box)
- Downloadable vocabulary lists

**Critical Rule:** **NO COGNATES**  
Test each word: "Would a Spanish speaker already guess this?" If YES → Don't teach it.

### **DOWNLOAD 3: PRONUNCIATION GUIDE (PDF)**

```
FORMAT: Pronunciation_Guide_[Lesson_Name].pdf

CONTENTS:
├─ Each target word with:
│  ├─ IPA notation
│  ├─ Syllable breakdown
│  ├─ Stress marking (PRIMARY stress in CAPS)
│  ├─ Common mispronunciation by Spanish speakers
│  └─ Correction tip
├─ Audio reference (MP3 links or QR codes)
└─ Practice sentences with stress patterns marked

EXAMPLE:
Word: interview
IPA: /ˈɪntərvjuː/
Syllables: IN-ter-view
Stress: INterview (first syllable)
Spanish speaker alert: Don't say "EEN-ter-view"
Practice: "I have an INterview tomorrow."
```

## ✅ VOCABULARY GLOSSARY CHECKLIST

**Before Publishing Any Vocabulary Lesson:**

- [ ] 8-12 target words identified (NO COGNATES)
- [ ] Each word includes:
  - [ ] IPA pronunciation
  - [ ] Part of speech
  - [ ] Clear definition
  - [ ] Spanish translation (A1-A2)
  - [ ] 2-3 HR context examples
  - [ ] 2-3 collocations
  - [ ] Common mistake alert (if applicable)
- [ ] Word families & relationships mapped
- [ ] False friends identified (Spanish speakers)
- [ ] Images/icons for each word (IMG-L##-VOC##)
- [ ] 3 interactive games created:
  - [ ] Vocabulary matching (8-10 pairs)
  - [ ] Word order challenge (6-8 sentences)
  - [ ] Fill-in-the-blank (8 questions)
- [ ] Games tested and functional
- [ ] Downloadable materials created:
  - [ ] Vocabulary list (PDF)
  - [ ] Flashcard set (PDF/Anki)
  - [ ] Pronunciation guide (PDF)
- [ ] Audio recordings (Web Speech API)
- [ ] All examples from HR professional contexts
- [ ] Mobile responsive design
- [ ] Accessibility features (alt text, ARIA labels)

## 🎨 DESIGN SPECIFICATIONS

**Visual Elements:**
- Consistent icon usage (🔊 audio, 📖 word, 💡 definition, etc.)
- Color coding: Vocab = blue accents, Grammar = orange accents
- Clear visual hierarchy
- Images support meaning (not decorative)

**Layout:**
- Clean, scannable format
- Generous white space
- Print-friendly margins
- Mobile-optimized spacing

---

## 🔄 REUSABILITY & SCALING

**Adapt for Levels:**
- A1-A2: Include Spanish throughout, simpler definitions
- B1-B2: Spanish for key terms only, more complex collocations
- C1-C2: English only, advanced usage notes, register variations

**Module Integration:**
- Create cumulative vocabulary lists per module
- Track word frequency across lessons
- Build on previously learned vocabulary
- Create vocabulary progression maps
